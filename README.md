# Crohns scRNA inflammation prediction
Single-cell RNA-seq (scRNA-seq) analysis pipeline using Scanpy with downstream ML models (PCA, kNN, XGBoost) to predict inflammation status in Crohn’s disease biopsies and identify key gene signatures.
